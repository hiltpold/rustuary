# Reserving workbench product notes

## Primary users

- Reserving actuaries
- Actuarial analysts
- Finance reviewers
- Model governance reviewers
- Engineering/platform operators

## Core user workflow

1. Load triangle and exposure data.
2. Validate data quality.
3. Review link ratios and development patterns.
4. Configure candidate methods.
5. Compare candidate ultimates and reserves.
6. Select methods by accident year.
7. Add rationale for overrides and blends.
8. Approve and export results.
9. Preserve audit trail.

## Product principle

The workbench should not hide actuarial judgment. It should make judgment easier to review.
